BlackBoxScanner Documentation

Installation

pip install blackbox-scanner

Classes Overview

1. ScannedOutputs

A utility class for storing and manipulating model outputs.

Key Methods:

  • add_token_output(token, output): Add a token and its corresponding output.
  • get_total(): Calculate the total of all outputs.
  • get_tokens(): Retrieve all tokens and their outputs.
  • get_perplexity(): Compute the average output value.

2. GenerativeModelOutputs

Analyze generative language model outputs.

Methods:

  • sentence_log_likelihoods(words): Compute negative log-likelihoods for tokens 
    context = "The furry pet is a ..."
output = gmo.sentence_log_likelihoods(context)
# Returns ScannedOutputs with token probabilities
  • view_topk(input_sentence, k, get_plot=False): Retrieve top-k probable next tokens 
    top_tokens = gmo.view_topk("Hello world", k=5)
# Returns dictionary of top tokens and scores

3. EmbeddingOutputs

Analyze and visualize word embeddings for encoder models.

Key Method:

  • get_embeddings_output(words_list): Extract normalized word embeddings 
    words = ["machine", "learning"]
embeddings = eo.get_embeddings_output(words)
# Returns ScannedOutputs with word embeddings
